ghcomment
=========

A simple CLI tool for commenting on GitHub PRs.

Usage
-----

```
Usage of ghcomment:

  -comment string
        A PR Comment
  -pr int
        A Pull Request number
  -repo string
        A GitHub repository on the format <org>/<repository>, or the local git repo if empty
  -signkey string
        A key used to create the signature
  -token string
        A github token
```
